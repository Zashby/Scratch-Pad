Paths to Travel

There are many paths you can follow into the world of programming languages and their applications. Below are just a few of the broader choices from those paths. Each has their own respective strengths and weaknesses, but all will leverage your problem solving skills to complete tasks. One of the best things you can do when following any of these paths is to keep an open mind, and never be afraid to learn new things.


Job Title Job Description Skills / Languages Example Median Earning
Web Developer Develop websites and web based applications HTML, CSS, JavaScript, JS Frameworks, SQL, Backend management Facebook, Twitter ~$80,000

Internet of Things Developer Develop hardware and software solutions to everyday situations Python, IoT Networking concepts, electronics, Arduino, Raspberry Pi Google Home, Alexa ~$80,000
Game Developer Develop game logic to include building engines required for rendering gaming world C#, Java, Unreal, Unity, Halo, Skyrim, Legend of Zelda, Hyperlight Drifter ~$91,000
