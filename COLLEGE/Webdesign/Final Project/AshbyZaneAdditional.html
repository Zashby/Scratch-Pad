Additional Resources



Sometimes it can seem difficult to figure out where to go to learn about such a complex and broad topic as programming. Just like music, there are many styles to learn and while it is easy to just make noise it can be difficult to make a song. None of these resources will do the coding for you, but they will help give you the tools necessary to move forward in your programming journey.


w3schools

A resource with basic tutorials on multiple modern programming languages


Codewars

Practice your coding skills with daily challenges in a variety of languages


FreeCodeCamp

Programming language platform focused on web development, backend management, and data science applications


EdX Statistcal Learning with R

A Stanford University level college class explaining statistical modeling and its application in the R language
